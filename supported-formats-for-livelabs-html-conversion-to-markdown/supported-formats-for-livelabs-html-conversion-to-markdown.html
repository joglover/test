HTML Entities

This topic applies only if your code block consists of HTML code. You do not need to do this for other code types, like sql and bash.

If you are writing HTML code in an HTML macro, you need to escape some of the HTML characters in your code by using HTML entities. Otherwise, Confluence interprets your HTML code as HTML, and tries to render it when you click Save.

For example, <html> should be written as

&lt;HTML&gt;

  • &lt; is used instead of <
  • &gt; is used instead of >

For a list of HTML entities, please see HTML Entities on w3school.com.
